# LaporAja

LaporAja adalah aplikasi pelaporan masyarakat berbasis web yang membantu pengguna melaporkan kerusakan fasilitas umum secara cepat, mudah, dan transparan.

---

## Problem

Banyak masyarakat kesulitan menyampaikan laporan kerusakan fasilitas umum secara efektif. Oleh karena itu, LaporAja dibuat untuk mempermudah proses pelaporan secara online dan membantu pengelolaan laporan oleh admin.

---

## Fitur Utama

- Login & autentikasi admin
- Upload laporan masyarakat
- Upload dokumentasi gambar
- Dashboard admin
- Filter & pencarian laporan
- Responsive mobile design
- Validasi form
- Middleware proteksi admin
- Optimistic UI & skeleton loading

---

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- Zod
